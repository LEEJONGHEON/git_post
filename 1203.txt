일치선택자 : 일치하는선택자 하나를 명시하여 적용
ex) li.test : li태그의 클래스가 test인것 (뛰어쓰기없이사용)

자식선택자 : 해당 태그의 바로 아래의 속성값 에 적용(자식 한단계만 적용)
ex) ul > .orang : ul태그아래의 orang 클래스에적용
<ul> <li class="orang"> 에만적용 ( >사용 )


후손선택자 : 해당 태그 아래의 모든 조건에 적용(아래 여러단계에 적용)
ex) div .orang : div 태그 후손중 .orang (뛰어쓰기)

인접형제 : 현재 요소 뒤로 뒤에 나오는 형제 요소 한개선택
조건 + 적용할 태그 : 해당조건을 만족할 태그 에만 적용
ex) li + li li 뒤 li에 적용(+ 사용)

일반형제 : 현재 요소 뒤로 뒤에 나오는 형제 요소 전체
ex) .orang ~ li : orang요소 뒤에 나오는 형제 요소 전체중 li 태그에적용
